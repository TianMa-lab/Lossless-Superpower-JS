# 飞书应用"悟空"配置指南

## 问题描述
飞书应用"悟空"的各个组件（机器人、网页应用、工作台小组件、云文档小组件）存在必填项未填写的问题。

## 配置步骤

### 1. 机器人组件配置

**必填项：**
- **机器人名称**：悟空
- **机器人头像**：上传一个合适的头像
- **机器人描述**：项目沟通机器人，用于项目管理和文档协作
- **消息卡片模板**：设置默认消息卡片样式
- **事件订阅**：根据需要订阅相关事件

**操作步骤：**
1. 登录飞书开发者平台
2. 进入"悟空"应用的控制台
3. 点击左侧菜单中的"机器人"选项
4. 填写所有必填项
5. 点击"保存"按钮

### 2. 网页应用组件配置

**必填项：**
- **应用名称**：悟空
- **应用图标**：上传一个合适的图标
- **应用描述**：项目沟通和文档管理应用
- **首页地址**：设置应用的首页URL
- **重定向地址**：设置OAuth重定向地址
- **权限范围**：选择需要的权限

**操作步骤：**
1. 进入"网页应用"选项
2. 填写所有必填项
3. 点击"保存"按钮

### 3. 工作台小组件配置

**必填项：**
- **小组件名称**：悟空工作台
- **小组件图标**：上传一个合适的图标
- **小组件描述**：项目管理工作台
- **小组件版本**：设置版本号（如1.0.0）
- **更新类型**：选择更新类型
- **基础信息**：填写小组件的基础信息

**操作步骤：**
1. 进入"工作台小组件"选项
2. 填写所有必填项
3. 点击"保存"按钮

### 4. 云文档小组件配置

**必填项：**
- **小组件名称**：悟空文档助手
- **小组件图标**：上传一个合适的图标
- **小组件描述**：文档协作助手
- **小组件版本**：设置版本号（如1.0.0）
- **更新类型**：选择更新类型
- **基础信息**：填写小组件的基础信息

**操作步骤：**
1. 进入"云文档小组件"选项
2. 填写所有必填项
3. 点击"保存"按钮

## 权限设置

确保为应用添加以下权限：

**必需权限：**
- `docx:document:readonly` - 读取文档内容
- `drive:comment:readonly` - 读取评论
- `drive:comment:write` - 写入评论

**可选权限：**
- `im:message:send` - 发送消息
- `im:chat:readonly` - 读取聊天信息

## 版本发布

完成所有配置后，创建一个新的版本并发布：

1. 进入"版本管理与发布"选项
2. 点击"创建版本"
3. 填写版本号（如1.0.5）
4. 填写更新说明
5. 点击"保存"按钮
6. 点击"发布"按钮

## 测试验证

使用以下命令测试飞书应用配置：

```bash
node check_feishu_connection.js
node test_feishu_skills.js
node test_feishu_chat.js
```

## 常见问题

1. **权限不足**：确保已添加所有必要的权限
2. **配置错误**：检查所有必填项是否已填写
3. **版本未发布**：确保已发布最新版本
